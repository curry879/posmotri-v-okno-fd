# Посмотри в окно
## О проекте

«Посмотри в окно» — это сайт, где можно смотреть видео из разных городов.

На сайте можно:
  - искать видео по городу;
  - выбирать время суток;
  - смотреть найденные видео;
  - загружать больше карточек;
  - видеть загрузку и ошибки. 

В этой работе нужно было стилизовать готовую разметку и сохранить работу JavaScript.

## Технологии

В проекте использовались:
  - HTML
  - CSS
  - JavaScript
  - БЭМ
